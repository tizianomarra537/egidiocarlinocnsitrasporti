# Construction Transport Website

## Overview
This project is a construction transport website that provides information about various transport services offered by the company. It includes a homepage, services page, about page, and contact page, all built using React.

## Project Structure
```
construction-transport-website
├── public
│   ├── index.html          # Main HTML document for the website
│   ├── favicon.ico         # Favicon for the website
│   └── assets
│       └── styles.css      # CSS styles for the website
├── src
│   ├── components
│   │   ├── Header.js       # Navigation bar and site title
│   │   ├── Footer.js       # Footer section of the website
│   │   └── TransportCard.js # Component to display transport services
│   ├── pages
│   │   ├── Home.js         # Homepage of the website
│   │   ├── Services.js     # Lists transport services
│   │   ├── About.js        # Information about the company
│   │   └── Contact.js      # Contact form for inquiries
│   ├── App.js              # Main application component
│   └── index.js            # Entry point for the React application
├── package.json            # npm configuration file
├── .gitignore              # Files and directories to ignore by Git
└── README.md               # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd construction-transport-website
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

- Information about transport services
